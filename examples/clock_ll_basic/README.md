# Basic Clock (LL API)

Простейший пример цифровых часов, использующий **внутренний RTC** (Real Time Clock) микроконтроллера RP2040.

## Описание
Пример демонстрирует минимально необходимый код для создания полезного устройства на базе библиотеки `display_ll`. 
Здесь нет сложной графики или шрифтов — только прямая запись масок цифр в буфер драйвера.

**Функционал:**
*   Инициализирует время значением `12:00:00`.
*   Отображает часы и минуты в формате `HH.MM`.
*   Точка (Decimal Point) мигает каждую секунду.

##  Требования
*   Библиотека `hardware_rtc` (включена в `CMakeLists.txt`).
*   Подключение дисплея аналогично примеру `ll_test`.

##  Как это работает
1.  **RTC:** Используется аппаратный таймер RP2040 для точного отсчета секунд.
2.  **Main Loop:**
    *   Считывает время (`rtc_get_datetime`).
    *   Преобразует числа в битовые маски сегментов.
    *   Добавляет бит точки (`0x80`) к второму разряду, если секунда четная.
    *   Отправляет данные в `display_ll`.

> **Примечание:** При отключении питания время сбросится, так как у RP2040 нет входа для батарейки RTC (VBAT) на стандартной плате Pico, но логика работы сохраняется.
