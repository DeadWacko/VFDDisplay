
# High-Level Basic Clock

Минималистичный пример использования **HL API**.

Демонстрирует, как библиотека абстрагирует всю сложность управления дисплеем.
В отличие от LL-примера, здесь:
1.  Не нужно формировать битовые маски цифр вручную.
2.  Мигание точек (`:`) управляется драйвером автоматически (`display_set_dot_blinking`).
3.  Основной цикл требует вызова `display_process()`.

**Функции:**
*   Показывает время `12:xx`.
*   Точки мигают автоматически.
*   Яркость статическая.
