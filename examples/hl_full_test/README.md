# High-Level Full Showcase (v1.3.7)

Комплексный пример использования всех возможностей библиотеки `vfd_display`.
Демонстрирует работу API, управление приоритетами, оверлеи и новую систему конфигурации.

## Сценарий демо

1.  **Init:** Расширенная инициализация через `display_init_ex` с кастомным конфигом.
2.  **Boot Overlay:** Анимация проверки сегментов (0..9).
3.  **Clock Mode:**
    *   Отображение времени `HH:MM`.
    *   **Особенность:** Используется `display_set_dots_config` для включения *мигающего* разделителя на 2-м разряде.
4.  **Info Mode (Temp):**
    *   Отображение температуры (`24.5`).
    *   **Особенность:** Переключение разделителя в режим *Static* (горит постоянно).
5.  **Text & Marquee:**
    *   Slide In: Выезд текста "HELLO".
    *   Marquee: Прокрутка длинной строки. Точки отключаются.
6.  **System Overlays:**
    *   **WiFi:** Симуляция подключения (мигание `8888`).
    *   **NTP:** Симуляция синхронизации времени ("Змейка").
7.  **Visual Effects (Прозрачные):**
    *   **Pulse:** Дыхание яркости.
    *   **Heartbeat:** Двойной удар сердца.
    *   **Scanner (Matrix):** Эффект KITT с периодом 1 сек.
    *   *В это время часы продолжают идти на фоне.*
8.  **Structural Effects (Блокирующие):**
    *   **Glitch:** Цифровой шум.
    *   **Morph:** Превращение `COOL` -> `VFD`.
    *   **Dissolve:** Рассыпание текста.
9.  **Fade Cycle:** Плавное гашение экрана и рестарт.
