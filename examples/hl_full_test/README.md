

# High-Level Full Showcase (hl_full_test)

Комплексный пример использования всех возможностей библиотеки `vfd_display`.
Программа работает как циклическая демо-сцена, переключая режимы отображения.

## Сценарий демо

1.  **Boot Overlay:** При старте запускается анимация "перебора цифр" (имитация теста сегментов).
2.  **Clock:** Отображение времени `19:xx` с мигающей точкой.
3.  **Slide In:** Справа выезжает дата (`01.01`).
4.  **Marquee:** Бегущая строка с длинным текстом (например, IP адрес).
5.  **WiFi Overlay:** Симуляция прерывания на подключение (мигание `8888`), перекрывающее контент.
6.  **Visual Effects:**
    *   **Pulse:** Плавная пульсация яркости ("дыхание").
    *   **Wave:** Волна яркости пробегает по разрядам.
    *   **Scanner:** Эффект "KITT" (бегущий огонь).
7.  **Structural Effects:**
    *   **Glitch:** Случайная подмена битов в разрядах ("цифровой шум").
    *   **Morph:** Плавное превращение текста `GOOD` в `BYTE`.
    *   **Dissolve:** Побитовое рассыпание текста в пустоту.
8.  **Fade Out/In:** Плавное гашение и возврат к часам.
