# Contributing

Пока проект развивается как личный pet-project. Если вы хотите что-то
предложить или изменить:

1. Откройте issue на GitHub с описанием идеи или бага.
2. Оформите PR, если есть готовое решение.
3. При изменении API обновляйте `docs/API.md`.
4. При добавлении эффектов обновляйте `docs/FX.md`.

Код пишется на C11 под Pico SDK.
