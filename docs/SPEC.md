# VFD Display Driver — SPEC

Цель проекта — создать переиспользуемый модуль драйвера VFD-дисплея для RP2040
с двумя уровнями API:

- низкоуровневый (`display_ll`) — работа с сырым буфером сегментов и яркостью;
- высокоуровневый (`display`) — показ времени, чисел, текста и эффектов.

## Текущее состояние

- Монолитный драйвер в файле `display.c`.
- Пример использования в `VFDDisplay.c`.

## Ближайшие шаги

1. Выделить структуру репозитория (src/include/examples/docs/...).
2. Оформить public API в заголовках `display_ll.h` и `display.h`.
3. Постепенно рефакторить `display.c` в сторону двухуровневой архитектуры.
