# ARCHITECTURE

Проект разделён на уровни:

1. Low-Level (LL) — управление сегментами, буфером, яркостью, таймерами.
2. Core — базовая логика обновления дисплея.
3. High-Level (HL) — текст, числа, время, точки.
4. FX — эффекты анимаций.
5. Overlay — временные поверхностные эффекты (WiFi, NTP, Boot).
6. Examples — практическое использование.

В текущей версии весь код находится в одном файле display.c.
Рефакторинг будет выполнен постепенно.
