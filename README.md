# VFDDisplay — VFD драйвер для Raspberry Pi Pico / RP2040

Это проект драйвера вакуумно-люминесцентного индикатора (VFD) на базе RP2040.
Сейчас код представляет собой монолитный пример (`VFDDisplay.c` + `display.c`),
но в будущем проект будет развиваться до полноценной библиотеки с низко- и
высокоуровневым API.

## Статус

Проект в активной разработке. Архитектура и API могут меняться.
